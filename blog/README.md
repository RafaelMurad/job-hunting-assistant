# Building a Job Hunting Assistant: A Developer's Journey

**About this blog series**

I built a job hunting assistant to solve my own problem: applying to jobs is tedious and inefficient. This is the story of how I built it, what I learned about collaborating with AI, and why I made the technical decisions I did.

**Tech Stack:** Next.js 16, TypeScript, Prisma, multi-AI integration (Gemini/OpenAI/Claude)

**Stats:** 50 files changed, 10,458+ lines of code, documented every decision
